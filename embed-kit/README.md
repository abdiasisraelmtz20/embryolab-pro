# Embed Kit · EmbryoLab PRO

| Archivo | Plataforma |
|---|---|
| `odoo.html` | Odoo Web Learning |
| `wordpress.html` | WordPress (Gutenberg, Elementor) |
| `moodle-canvas.html` | Moodle, Canvas, Blackboard |

## Uso

1. Reemplaza `TU-USUARIO` por tu usuario de GitHub
2. Ajusta `height="900"` si el contenido queda cortado o sobra espacio
3. Pega en el editor HTML de la plataforma

## Requisito para AR

El atributo `allow` DEBE incluir `xr-spatial-tracking`.

## Plataformas con filtros HTML estrictos

- **Moodle:** Admin → Filtros → Filtros HTML → permitir `<iframe>`
- **WordPress:** instala plugin `iframe`
